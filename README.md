## 🌐 The Privacy Revolution

As Web3 adoption accelerates, users face a critical dilemma: their digital assets are decentralized, but their access layer remains highly centralized. Legacy VPNs demand emails, passwords, and fiat subscriptions, instantly compromising anonymity before you ever connect. 

**Solana Shield VPN** is purpose-built to eliminate the friction and vulnerability of traditional privacy tools. We replaced legacy authentication with the **Solana Mobile Wallet Adapter (MWA)**. 

> **We Know Nothing About You. And That’s The Point.** 
> By leveraging cryptographic signatures instead of traditional accounts, we have removed the single biggest point of failure in the privacy stack: the user database.

No emails. No passwords. No fiat. Just connect your wallet (Phantom, Solflare) and instantly access a globally distributed network of secure, high-speed nodes.

## 📱 The Product Experience

1. **Launch**: Open the app.
2. **Authenticate**: Tap "Connect Wallet". Securely sign the MWA request directly in your device widget.
3. **Secure**: Tap the glowing connection shield. Your traffic is instantly encrypted.

**Under 5 seconds from download to protected connection.**

## 🛠 Technical Architecture

Engineered from scratch for speed, security, and a buttery-smooth 120fps UI:
- **Client Application:** Robust cross-platform mobile client built via **Flutter / Dart**.
- **State Management:** Reactive and reliable architecture powered by **Riverpod**.
- **VPN Core:** Underlying custom routing engine optimized with gRPC and Protocol Buffers.
- **Identity Layer:** Deep integration with Solana Mobile Wallet Adapter.

## 📦 Getting Started

### Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install)
- Compatible Android Studio/Xcode configuration for Native builds
- Solana Wallet (e.g., Phantom App installed on test device/emulator)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository_url>
   cd Solana_Shield_VPN
   ```
2. Fetch dependencies:
   ```bash
   flutter pub get
   ```
3. Run the application:
   ```bash
   flutter run
   ```
